# Windows

Apesar da forte presença do Linux em servidores, grande parte das empresas utiliza Windows em estações de trabalho e ambientes corporativos.

Conhecer sua arquitetura e mecanismos de segurança é indispensável.

## O que você aprenderá

- Estrutura do Windows
- Registro
- Serviços
- Active Directory
- PowerShell
- Sistema de arquivos NTFS
- Gerenciamento de usuários

> **Objetivo:** compreender o funcionamento interno do Windows e seus recursos de segurança.
