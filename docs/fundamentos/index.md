# Fundamentos

Bem-vindo ao módulo de **Fundamentos**.

Antes de aprender ferramentas, técnicas ou realizar testes de segurança, é essencial compreender os conceitos que sustentam a Cybersecurity.

Neste módulo você desenvolverá a base necessária para entender como a segurança da informação funciona e como os profissionais da área analisam riscos, protegem ativos e tomam decisões.

## O que você aprenderá

- O que é Cybersecurity
- Tríade CIA
- Princípio do Menor Privilégio
- Ameaças
- Vulnerabilidades
- Riscos
- Controles de Segurança

> **Objetivo:** construir uma base sólida para compreender os módulos seguintes.
