# Redes

Todo profissional de Cybersecurity precisa entender como os dispositivos se comunicam.

Grande parte dos ataques e das defesas acontece através da rede. Por isso, compreender protocolos, endereçamento e modelos de comunicação é indispensável.

## O que você aprenderá

- Modelo OSI
- Modelo TCP/IP
- Endereçamento IPv4 e IPv6
- DNS
- DHCP
- NAT
- Portas
- Protocolos de Rede

> **Objetivo:** compreender como a comunicação entre dispositivos acontece na prática.
