# Linux

O Linux está presente em servidores, ambientes cloud, dispositivos embarcados e diversas ferramentas utilizadas em Cybersecurity.

Dominar seus comandos e estrutura é uma habilidade essencial para qualquer profissional da área.

## O que você aprenderá

- Sistema de arquivos
- Permissões
- Usuários e grupos
- Terminal
- Bash
- Processos
- Serviços
- Gerenciamento de pacotes

> **Objetivo:** desenvolver confiança para trabalhar com sistemas Linux no dia a dia.
